# Recent-News Claim Set

This folder is reserved for curated, time-sensitive claims where stale pretraining is a realistic risk.

The included examples are synthetic but formatted to match the intended workflow:

- a recent-news style source paragraph
- expected claim labels
- a separate corpus with supporting, refuting, or non-decisive evidence

For the final project, replace or extend these examples with manually curated real recent claims and verified source passages.
